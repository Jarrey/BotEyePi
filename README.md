# BotEyePi
----
**A "robot eye" detection for industry based on Python and Rasberry Pi platform**



## Environment
* Python 2.7

## Dependencies
* **pyhton-zbar**
	* (in Debian) `sudo apt-get install python-zbar`
* **python-imaging**
	* (in Debian) `sudo apt-get install python-imaging`


## Features
* Detect bar image:
	* `python barreader.py [image file path]`
